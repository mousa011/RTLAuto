From UI transformation to production-ready RTL in seconds.

RTLAuto is a Figma plugin that transforms designs between LTR and RTL with a faster, cleaner, and more production-ready workflow. Mirror layouts, handle BiDi text correctly, flip directional icons with context awareness, and translate text when needed — all inside Figma.

⚙️ Setup Install the plugin from the Figma Community Open your Figma file Run RTLAuto from Plugins → Development / Installed Plugins

✨ Features

Transform designs between LTR and RTL in both directions
Smart layout mirroring with context-aware icon flipping
BiDi-aware text handling for Arabic, English, and mixed content
Translate text with Google, Gemini, or OpenAI
Use your own API keys for AI translation providers
Create LTR/RTL variants for components and component sets
Duplicate and mirror frames safely
Choose fallback fonts and handle missing fonts more reliably
Free daily usage with paid plans for advanced workflows
🧩 Input Requirements

Select frames, components, component sets, or supported layers
Some actions work with a single selection, while others support multiple layers
Text-only translation can be used without changing layout
📦 Output Generate production-ready RTL or LTR design output directly in Figma, including:

Mirrored layouts
Updated text direction and alignment
Context-aware flipped icons
Translated text when enabled
RTL/LTR variants for supported components
🚀 How to Use

Select the design, frame, or component you want to transform
Run RTLAuto from Figma plugins
Choose the action you want, such as Smart Transform, Mirror, Translate, or Create Variants
Review the result inside Figma
Continue editing or hand off the output to development
🔒 Privacy Core transformations run locally inside Figma. Translation features use external providers only when enabled by the user. AI translation with Gemini or OpenAI uses the user’s own API keys.

⚠️ Notes

Results may vary depending on layer structure and text setup
Some advanced translation features require provider configuration
Fallback font behavior depends on fonts available in your Figma environment
